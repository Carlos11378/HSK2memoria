Um atlas interativo completo para o **HSK 2 (汉语水平考试二级)** com **15 lições**, **textos e diálogos reais**, **300 palavras**, **quiz interativo** e **prática de escrita** — tudo em um único arquivo HTML.

---

## 🐉 Sobre o Projeto

Este projeto foi desenvolvido para estudantes de chinês que estão se preparando para o **HSK 2**. Ele oferece uma experiência imersiva e completa de estudo, com:

- 📚 **15 lições** com vocabulário, frases e diálogos autênticos do curso padrão HSK 2.
- 🃏 **Flashcards interativos** com rotação 3D, navegação por teclado e progresso visual.
- 🎯 **Quiz** com perguntas aleatórias de múltipla escolha para testar seu conhecimento.
- ✍️ **Painel de escrita** para praticar traços dos caracteres (compatível com mouse e toque).
- 🔤 **Glossário completo** com todas as **300 palavras** do HSK 2, com busca rápida.
- 🎨 **Design imersivo** com tema vermelho e dourado, inspirado na cultura chinesa.

---

## ✨ Funcionalidades Detalhadas

| Funcionalidade | Descrição |
|---------------|-----------|
| **Modo Cartões** | Flashcards com frente/verso, animação 3D, navegação por teclado (← →) |
| **Modo Quiz** | Perguntas aleatórias com 4 opções, pontuação em tempo real |
| **Modo Escrita** | Desenhe os caracteres com o mouse ou toque, com grade de apoio |
| **Painel Lateral** | Lição completa com vocabulário, diálogos e busca |
| **Barra de Progresso** | Mostra o avanço geral no estudo das 300 palavras |
| **Atalhos de Teclado** | Navegue sem usar o mouse: ← → ↑ ↓ Espaço 1 2 3 |
| **Fundo Animado** | Caracteres chineses flutuando no fundo para imersão |
| **Responsivo** | Funciona em desktop, tablet e mobile |

---

## 🖥️ Tecnologias Utilizadas

- **HTML5** + **CSS3** (design responsivo e imersivo)
- **JavaScript puro** (sem frameworks ou dependências externas)
- **Canvas API** para o painel de escrita e animação de fundo
- **Google Fonts** (Noto Sans/Serif SC + Inter)
